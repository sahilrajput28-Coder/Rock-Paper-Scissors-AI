# Rock-Paper-Scissors AI (C++)

This is a smart Rock-Paper-Scissors game built using C++.

## Features
- User vs Computer gameplay
- AI predicts user moves using pattern recognition
- Multi-round game with score tracking

## How it works
The AI stores previous user moves in an array and uses frequency analysis to predict the next move. It then plays the counter move to win.

## Technologies Used
- C++
- Arrays
- Loops
- Conditional Statements
